# 🎮 **AP Game**  
**Professional Gaming Magazine & News Platform**

---

## 📖 **About the Project**

**AP Game** is a modern, bilingual (Persian/English) gaming magazine that provides users with the latest gaming news, top game reviews, weekly challenges, and a comment system. This project is designed to create a professional platform for the Iranian gaming community.

---

## ✨ **Key Features**

### 🏠 **Home Page**
- Animated banner slider with floating emojis
- Text slider showcasing different site sections
- Advanced comment system with reply functionality
- User registration with name, phone number, and avatar

### 📰 **News Section**
- Professional card-based news display
- Hot and new badges for each news item
- LocalStorage-based data persistence

### 🎮 **Games Section**
- Top 10 all-time best games list
- Game cards with images, genre, and summaries
- Genre-based color coding (Action, Adventure, RPG, etc.)
- Buy/Install buttons linking to official stores

### 🏆 **Weekly Challenges**
- Gaming interest quiz (10 questions)
- Number guessing game with rewards
- Results displayed in the rewards section

### 🎁 **Rewards Section**
- Display quiz results card
- Display game win card
- Exclusive rewards based on user interest level

### 👤 **User System**
- Registration with name, phone number, and avatar
- User data stored in browser localStorage
- User status displayed in the site header

### 🌐 **Bilingual Support**
- Full Persian and English language support
- One-click language switching
- RTL/LTR direction switching

---

## 🛠️ **Technologies Used**

- **HTML5**: Page structure
- **CSS3**: Styling and design (Dark theme with gold accents)
- **JavaScript**: Site logic, comment system, data storage
- **LocalStorage**: User data, comments, news, and games storage
- **GitHub Pages**: Free and permanent hosting

## 🚀 **Installation & Setup**

### Method 1: Local Execution
1. Download or clone the project.
2. Open `index.html` in your browser.
3. All features work without a server.

### Method 2: GitHub Pages Hosting
1. Push the repository to GitHub.
2. Enable Pages in repository settings.
3. Get your site link and share it.
